CSS (Cascading Style Sheets)

Css is for styling your html code. Using CSS you can connect your stylesheet to different tags such as "< div >, < nav >, and < section >. So far the tags that have helped me with positioning are Flexboxes and Absolute/Relative positioning. If you want your page to be responsive then you must use media queries(@media). This will allow a mobile or desktop to switch to a larger or smaller size smoothly without items overlapping and covering content.
